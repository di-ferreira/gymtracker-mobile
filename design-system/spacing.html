Escala de Espaçamento

Grid base de 4px. Cada token é múltiplo de 4 para manter a consistência visual e o alinhamento entre plataformas. Use os tokens em vez de valores livres.

TokenValorUso
--gt-space-00pxSem espaçamento
--gt-space-14pxMicro gaps, ícone-label
--gt-space-28pxGaps internos, padding de input
--gt-space-312pxGaps entre elementos, padding de card
--gt-space-416pxPadding de seção, gaps de lista
--gt-space-520pxPadding de card grande
--gt-space-624pxPadding de container, gaps de grid
--gt-space-832pxMargens de página, separadores
--gt-space-1040pxSeções de página
--gt-space-1248pxHeaders, separadores grandes
--gt-space-1664pxMargens de hero

Regras

  • Sempre use tokens de espaço — nunca valores livres em CSS ou RN StyleSheet
  • Use múltiplos de 4px para qualquer espaço não coberto pelos tokens
  • Padding e gap no mesmo componente devem usar tokens do mesmo nível ou adjacente
  • Mobile: reduzir space-8+ para space-6 em telas menores que 400px
  • Web: usar space-8 como padding mínimo de conteúdo em containers
