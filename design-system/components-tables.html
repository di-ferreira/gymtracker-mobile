Regras de Uso

  • Sempre use font-variant-numeric: tabular-nums em colunas numéricas para alinhamento vertical consistente
  • Header com texto uppercase e letter-spacing: 0.06em — nunca use title case
  • Tabelas compactas apenas em visualizações com 6+ colunas ou painéis de métricas densas
  • A barra de ações em lote só aparece quando há pelo menos 1 linha selecionada
  • O checkbox "selecionar todos" marca apenas as linhas da página atual, não todas as páginas
  • Estado vazio deve sempre exibir mensagem com call-to-action, nunca uma tabela vazia sem contexto
  • Colunas ordenáveis devem mostrar indicador visual (seta) no hover e na coluna ativa
  • Paginação obrigatória para tabelas com mais de 25 linhas — use scroll interno apenas em tabelas compactas
  • Nunca use zebra striping (alternância de fundo) — o hover é suficiente para guiar o olhar
  • Em mobile, tabelas devem ter scroll horizontal com a primeira coluna fixa (sticky)
  • Chips de status usam cores semânticas: --gt-success para Concluído, --gt-warning para Pendente

Faça e Não Faça

Faça

Use tabular-nums em colunas numéricas para alinhar dígitos verticalmente.

Use chips de status com cores semânticas para comunicar o estado de cada linha.

Use paginação para conjuntos grandes de dados (> 25 linhas).

Não Faça

Não use zebra striping (fundo alternado) — dificulta a leitura e polui visualmente.

Não ordene colunas sem indicador visual — o usuário precisa saber qual coluna está ativa.

Não exiba mais de 50 linhas sem paginação em tabelas default.

Copie o bloco abaixo para o seu projeto.

:root {
  /* Table Header */
  --gt-table-header-bg: var(--gt-surface);             /* #151515 */
  --gt-table-header-fg: var(--gt-fg-muted);            /* #555555 */
  --gt-table-header-font: 13px;
  --gt-table-header-transform: uppercase;
  --gt-table-header-spacing: 0.06em;

  /* Table Rows */
  --gt-table-row-height: 48px;
  --gt-table-row-height-compact: 36px;
  --gt-table-row-border: 1px solid var(--gt-border);  /* rgba(255,255,255,0.08) */
  --gt-table-row-hover-bg: var(--gt-surface-2);         /* #1D1D1D */

  /* Selected Row */
  --gt-table-selected-bg: rgba(255, 122, 26, 0.06);
  --gt-table-selected-border: 2px solid var(--gt-primary); /* #FF7A1A */

  /* Checkbox */
  --gt-table-checkbox-size: 18px;
  --gt-table-checkbox-border: 1.5px solid var(--gt-border-strong);
  --gt-table-checkbox-checked-bg: var(--gt-primary);  /* #FF7A1A */

  /* Sortable */
  --gt-table-sort-arrow: ▲;
  --gt-table-sort-active-fg: var(--gt-primary);      /* #FF7A1A */

  /* Pagination */
  --gt-table-page-btn-size: 32px;
  --gt-table-page-btn-radius: 6px;
  --gt-table-page-btn-active-bg: var(--gt-primary); /* #FF7A1A */

  /* Numeric Cols */
  --gt-table-num-font: var(--gt-font-mono);            /* JetBrains Mono */
  --gt-table-num-variant: tabular-nums;

  /* Status Chips */
  --gt-table-chip-concluido-bg: var(--gt-success-surface); /* rgba(34,197,94,0.12) */
  --gt-table-chip-concluido-fg: var(--gt-success);   /* #22C55E */
  --gt-table-chip-pendente-bg: var(--gt-warning-surface);  /* rgba(245,158,11,0.12) */
  --gt-table-chip-pendente-fg: var(--gt-warning);  /* #F59E0B */
}
