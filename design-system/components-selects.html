Mobile vs Web

O componente seletor se comporta de forma diferente em cada plataforma para respeitar as convenções nativas.

Mobile

No app React Native, use um ActionSheet (iOS) ou bottom sheet picker (Android) para selecionar opções. Evite dropdowns flutuantes — eles não são ergonômicos em telas touch. O trigger permanece visualmente igual, mas ao tocar, abre o sheet de baixo para cima com as opções em lista scrollável.

  • Use no React Native
  • O sheet ocupa no máximo 50% da tela
  • Opções com altura mínima de 48px para toque
  • Adicione gesto de swipe-down para fechar

Web

No dashboard Next.js, use o dropdown customizado com painel absoluto. O dropdown posiciona-se abaixo do trigger com 4px de gap. Suporta teclado: ↑ ↓ para navegar, Enter para selecionar, Esc para fechar.
