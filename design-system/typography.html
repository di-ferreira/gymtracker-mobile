Famílias Tipográficas

O GymTracker usa duas famílias principais: Inter para display e body, JetBrains Mono para código e dados tabulares. A mesma família (Inter) é usada em display e body com peso e rastreamento diferentes para estabelecer hierarquia.

Display & Body
Inter
Aa Bb Cc
Pesos: 400 (Read), 500 (Emphasize), 600–700 (Announce)
Fallback: -apple-system, BlinkMacSystemFont, system-ui, sans-serif
Mono
JetBrains Mono
0O 1lI
Pesos: 400, 500
Fallback: IBM Plex Mono, ui-monospace, Menlo, monospace
Usado para: código, IDs, dados tabulares, timestamps
Numeros
Tabular Nums
1234567890
font-variant-numeric: tabular-nums
Usado para: métricas, pesos, séries, estatísticas

Sistema de Pesos

Três pesos de uso para manter a disciplina tipográfica em toda a interface.

400 Read Corpo de texto, parágrafos, descrições
500 Emphasize Labels, navegação, UI text, botões médios
600–700 Announce Headlines, CTAs, títulos, números de destaque

Regras de Rastreamento

Letter-spacing é obrigatório e não opcional. Títulos sem rastreamento negativo parecem soltos; ALL CAPS sem rastreamento positivo parecem amadores.

  • Display (48px+): -0.03em — obrigatório para evitar que o texto pareça solto
  • H1–H3 (20–32px): -0.01em a -0.02em — rastreamento negativo consistente
  • Body (14–16px): 0 — nunca adicione rastreamento em corpo de texto
  • Small / Caption (11–12px): 0.01em a 0.02em — rastreamento positivo sutil
  • ALL CAPS / Overline: 0.06em a 0.1em — piso de 0.06em para evitar que as letras colem

Regras de Comprimento de Linha

Limite o corpo de texto a 50–75 caracteres por linha para legibilidade ideal.

  • Corpo de texto: max-width: 65ch
  • Labels e UI text: sem limite de linha
  • Nunca use text-align: justify — cria rios de espaço no texto

Tokens CSS

Copie o bloco abaixo para o seu projeto.

:root {
  --gt-font-display: 'Inter', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
  --gt-font-body: 'Inter', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
  --gt-font-mono: 'JetBrains Mono', 'IBM Plex Mono', ui-monospace, Menlo, monospace;

  --gt-text-display: 48px / 1.05 / 700 / -0.03em;
  --gt-text-h1: 32px / 1.1 / 700 / -0.02em;
  --gt-text-h2: 24px / 1.2 / 600 / -0.01em;
  --gt-text-h3: 20px / 1.3 / 600 / -0.01em;
  --gt-text-body: 16px / 1.5 / 400 / 0;
  --gt-text-body-sm: 14px / 1.5 / 400 / 0.01em;
  --gt-text-caption: 12px / 1.5 / 500 / 0.02em;
  --gt-text-overline: 11px / 1.5 / 600 / 0.08em;

  --gt-weight-read: 400;
  --gt-weight-emphasize: 500;
  --gt-weight-announce: 700;
}
