Mobile vs Web

Diferenças de comportamento e layout do card em cada plataforma.

Mobile

No app React Native, cards ocupam a largura total da tela com padding lateral de 16px. A altura mínima do alvo tátil é 44px. Workout Cards inline devem mostrar no máximo 3 dados por linha para evitar overflow. Use TouchableWithoutFeedback + Animated para o estado pressed com scale.

  • Largura 100% com padding lateral space-4 (16px)
  • Altura mínima de toque: 44px
  • Máximo 3 pontos de dados por linha
  • Use TouchableWithoutFeedback + Animated.scale
  • Swipe lateral no Workout Card para revelar ações

Web

No dashboard Next.js, cards usam grid layout com largura máxima de 400px. O hover é ativado por :hover com transição suave de borda e sombra. Cards interativos devem ter tabindex="0" e simular hover com :focus-visible para acessibilidade de teclado.

  • Largura máxima 400px em grid responsivo
  • Hover via :hover + :focus-visible
  • tabindex="0" em cards interativos
  • Transição de hover com var(--gt-duration-normal)
  • Elevação progressiva elevation-1 → elevation-2

Regras de Uso

  • Card padding: 16px (space-4) padrão, 20px (space-5) para large
  • Card border-radius: 12px (radius-lg)
  • Card border: 1px solid rgba(255,255,255,0.08) padrão
  • Card elevation on hover: elevation-2
  • Altura mínima do alvo tátil: 44px no mobile
  • Hierarquia de conteúdo: título (semibold 15px) → metadados (13px secondary) → ação (cor primária)
  • Nunca empilhar mais de 3 pontos de dados em uma única linha do card
  • Use --gt-primary apenas para links de ação, nunca para bordas decorativas
  • Cards disabled devem usar opacidade 50% e pointer-events: none
  • O estado pressed deve usar scale(0.98) com duração rápida (100ms)
  • Nunca use mais de 2 cards por linha em telas menores que 768px
  • A transição de hover do card deve usar --gt-duration-normal (200ms) com --gt-ease-standard
  • Cards de estatísticas (Stat Card) devem sempre exibir o rótulo abaixo do valor, nunca ao lado
  • Progress Card deve sempre incluir a barra de progresso quando houver meta definida

Exemplo de Código

Uso do Card com React Native e tokens CSS.

// React Native — WorkoutCard.tsx
import { View, Text, Pressable } from 'react-native';
import { tokens } from '@gymtracker/tokens';

interface WorkoutCardProps {
  variant: 'elevated' | 'outlined' | 'filled';
  padding: 'sm' | 'md' | 'lg';
  interactive: boolean;
  disabled: boolean;
  platform: 'mobile' | 'web' | 'both';
}

export function WorkoutCard({ variant, padding, interactive, disabled }: WorkoutCardProps) {
  const padMap = { sm: tokens.space[3], md: tokens.space[4], lg: tokens.space[5] };
  const bgMap  = { elevated: tokens.surface, outlined: tokens.surface, filled: tokens.surface2 };

  return (
    <Pressable
      disabled={disabled}
      style={[{
        backgroundColor: bgMap[variant],
        borderRadius: tokens.radiusLg,     // 12px
        padding: padMap[padding],            // 16px (md)
        borderWidth: 1,
        borderColor: 'rgba(255,255,255,0.08)',
        opacity: disabled ? 0.5 : 1,
        minHeight: platform !== 'web' ? 44 : undefined,
      }, variant === 'elevated' && tokens.elevation[1]]}
    >
      <Text style={{ fontSize: 15, fontWeight: '600', color: tokens.fg }}>
        Supino Reto
      Text>
      <Text style={{ fontSize: 13, color: tokens.fgSecondary }}>
        4×12 · 60kg · 90s descanso
      Text>
      <Text style={{ fontSize: 13, color: tokens.primary, fontWeight: '500' }}>
        Ver detalhes →
      Text>
    Pressable>
  );
}

/* CSS Tokens para cards */
.gt-card {
  background: var(--gt-surface);                 /* #151515 */
  border: 1px solid rgba(255,255,255,0.08);
  border-radius: var(--gt-radius-lg);           /* 12px */
  padding: var(--gt-space-4);               /* 16px */
  transition: all 200ms var(--gt-ease-standard);
}
.gt-card--interactive:hover {
  border-color: rgba(255,122,26,0.25);
  box-shadow: var(--gt-elevation-2), 0 0 0 1px rgba(255,122,26,0.08);
}
.gt-card--interactive:active {
  transform: scale(0.98);
  transition-duration: 100ms;
}
