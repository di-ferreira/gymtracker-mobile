Fundamentos

Cores

Paleta de cores do tema escuro do GymTracker. Cada token inclui valores HEX e OKLch para consistência entre plataformas.

Paleta Primária

A identidade visual do GymTracker. Usada para CTAs, indicadores ativos e destaques.

--gt-primary #FF7A1A oklch(72% 0.175 55)
--gt-primary-hover #FF9249 oklch(77% 0.16 55)
--gt-primary-active #E06510 oklch(65% 0.19 50)

Superfícies

Hierarquia de camadas para o tema escuro. Cada nível eleva o conteúdo acima do background.

--gt-bg #0A0A0A oklch(12% 0.01 0)
--gt-surface #151515 oklch(18% 0.01 0)
--gt-surface-2 #1D1D1D oklch(22% 0.01 0)
--gt-surface-3 #252525 oklch(27% 0.01 0)
--gt-surface-4 #2D2D2D oklch(31% 0.01 0)

Texto

Níveis de hierarquia textual. Do principal ao silenciado.

--gt-fg #FFFFFF oklch(100% 0 0)
--gt-fg-secondary #B8B8B8 oklch(76% 0.01 0)
--gt-fg-tertiary #888888 oklch(58% 0.01 0)
--gt-fg-muted #555555 oklch(39% 0.01 0)

Cores Semânticas

Estados de feedback. Cada cor possui uma variante de superfície com 12% de opacidade para fundos leves.

--gt-success #22C55E · rgba(34, 197, 94, 0.12) Success · Success Surface
--gt-warning #F59E0B · rgba(245, 158, 11, 0.12) Warning · Warning Surface
--gt-error #EF4444 · rgba(239, 68, 68, 0.12) Error · Error Surface
--gt-info #3B82F6 · rgba(59, 130, 246, 0.12) Info · Info Surface

Bordas

Bordas semi-transparentes. Branco com opacidade para tema escuro — nunca use bordas escuras sólidas.

--gt-border rgba(255, 255, 255, 0.08) Borda padrão
--gt-border-strong rgba(255, 255, 255, 0.15) Borda forte (hover, foco)

Contraste

Razões de contraste WCAG 2.1 para combinações de texto sobre fundo. Mínimo exigido: 4.5:1 (AA).

Combinação Razão Resultado
#FFFFFF sobre #0A0A0A 18.05:1 Aprovado
#B8B8B8 sobre #0A0A0A 9.68:1 Aprovado
#888888 sobre #0A0A0A 5.21:1 Aprovado
#FFFFFF sobre #151515 14.62:1 Aprovado
#FFFFFF sobre #1D1D1D 12.57:1 Aprovado
#FF7A1A sobre #0A0A0A 5.12:1 Aprovado

Regras do Tema Escuro

Princípios obrigatórios para manter a coerência visual no tema escuro do GymTracker.

Atenção: Estas regras devem ser seguidas em todas as plataformas (React Native e Next.js). Violações causam inconsistência visual e problemas de acessibilidade.

Regra Descrição
Sem preto ou branco puro Nunca use #000000 como fundo ou #FFFFFF como cor de fundo. Use os tokens de superfície e texto.
Bordas semi-transparentes Use branco com opacidade (rgba(255,255,255,0.08)) para bordas. Nunca use bordas escuras sólidas.
Primary é a única cor saturada Sobre uma superfície, a cor primária (#FF7A1A) deve ser o único elemento saturado visible. Cores semânticas apenas para feedback.
Superfícies semânticas a 12% Cores semânticas usam fundos com 12% de opacidade da cor base, nunca a cor pura como fundo.
Hierarquia de superfícies bg → surface → surface-2 → surface-3 → surface-4. Cada nível representa uma camada visual acima da anterior.
Orçamento de destaque No máximo 2 usos visíveis da cor primária por tela. O excesso dilui o impacto visual.

Bloco de custom properties para copiar e usar no projeto.

:root {
  /* Paleta Primária */
  --gt-primary: #FF7A1A;          /* oklch(72% 0.175 55) */

  /* Superfícies */
  --gt-bg: #0A0A0A;              /* oklch(12% 0.01 0) */
  --gt-surface: #151515;          /* oklch(18% 0.01 0) */
  --gt-surface-2: #1D1D1D;       /* oklch(22% 0.01 0) */
  --gt-surface-3: #252525;        /* oklch(27% 0.01 0) */
  --gt-surface-4: #2D2D2D;       /* oklch(31% 0.01 0) */

  /* Texto */
  --gt-fg: #FFFFFF;              /* oklch(100% 0 0) */
  --gt-fg-secondary: #B8B8B8;    /* oklch(76% 0.01 0) */
  --gt-fg-tertiary: #888888;     /* oklch(58% 0.01 0) */
  --gt-fg-muted: #555555;        /* oklch(39% 0.01 0) */

  /* Bordas */
  --gt-border: rgba(255, 255, 255, 0.08);
  --gt-border-strong: rgba(255, 255, 255, 0.15);

  /* Semânticas */
  --gt-success: #22C55E;
  --gt-success-surface: rgba(34, 197, 94, 0.12);
  --gt-warning: #F59E0B;
  --gt-warning-surface: rgba(245, 158, 11, 0.12);
  --gt-error: #EF4444;
  --gt-error-surface: rgba(239, 68, 68, 0.12);
  --gt-info: #3B82F6;
  --gt-info-surface: rgba(59, 130, 246, 0.12);
}
