Durações

A escala de duração define os tempos de animação. Durações menores que 100ms não são percebidas; maiores que 500ms causam sensação de lentidão.

TokenValorUso
--gt-duration-instant0msSem transição (imediato)
--gt-duration-fast100msCor, opacidade, borda — mudanças de estado
--gt-duration-normal200msTransform, posição — mudanças de layout
--gt-duration-slow300msExpansão, colapso — sidebar, drawer
--gt-duration-slower500msPage transitions, modais complexos

Curvas de Easing

Curvas nomeadas para movimento consistente entre plataformas. Cada curva comunica uma intenção diferente.

Standard
cubic-bezier(0.25, 0.1, 0.25, 1)
Transições gerais, hover, foco, mudanças de cor
Decelerate
cubic-bezier(0, 0, 0.2, 1)
Elementos que entram: slides, sheets, modais
Accelerate
cubic-bezier(0.4, 0, 1, 1)
Elementos que saem: dismiss, close, fade-out
Spring
cubic-bezier(0.34, 1.56, 0.64, 1)
Micro-interações: like, check, contador
Sharp
cubic-bezier(0.4, 0, 0.6, 1)
Transações rápidas: toggle, press feedback

Transições Compartilhadas (React Native)

Diretrizes para transições de elemento compartilhado entre telas no app React Native.

  • Use SharedTransition do Reanimated para elementos que mudam de posição entre telas (ex: card → detalhe)
  • Duração: 300ms com curva decelerate para entradas e accelerate para saídas
  • Elementos compartilhados: imagem do exercício, avatar, card de treino
  • Não compartilhe mais de 2 elementos simultaneamente entre duas telas
  • Use ScreenTransition do React Navigation para page transitions: fade + slide em iOS, fade em Android

Regras

  • Reduza movimento quando o usuário ativar "Reduce Motion" (prefers-reduced-motion) — use 0ms ou fade apenas
  • Não use animações decorativas sem propósito — cada animação comunica uma mudança de estado
  • Máximo de 2 animações simultâneas por componente (ex: opacity + transform)
  • Mobil: animações de 300ms+ devem ser interruptíveis (gesture-driven)
  • Web: use CSS transitions para mudanças de estado; reserve JS animations para sequências complexas

Tokens CSS

Copie o bloco abaixo para o seu projeto.

:root {
  --gt-duration-instant: 0ms;
  --gt-duration-fast: 100ms;
  --gt-duration-normal: 200ms;
  --gt-duration-slow: 300ms;
  --gt-duration-slower: 500ms;

  --gt-ease-standard: cubic-bezier(0.25, 0.1, 0.25, 1);
  --gt-ease-decelerate: cubic-bezier(0, 0, 0.2, 1);
  --gt-ease-accelerate: cubic-bezier(0.4, 0, 1, 1);
  --gt-ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
  --gt-ease-sharp: cubic-bezier(0.4, 0, 0.6, 1);
}

@media (prefers-reduced-motion: reduce) {
  * { transition-duration: 0ms !important; animation-duration: 0ms !important; }
}
