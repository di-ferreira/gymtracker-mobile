# Repositório: gymtracker-mobile

## Objetivo

Aplicativo mobile para consulta de exercícios físicos durante o treino.

O aplicativo deve permitir:

- Navegar pelo catálogo
- Buscar exercícios
- Filtrar exercícios
- Consultar detalhes
- Consultar substituições
- Criar treinos personalizados
- Registrar cargas utilizadas
- Registrar histórico de execução
- Funcionar offline
